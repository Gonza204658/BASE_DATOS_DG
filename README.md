# RiwiSupply - Base de Datos

## Descripción

Este proyecto consiste en el diseño e implementación de una base de datos para RiwiSupply utilizando PostgreSQL. A partir del archivo de Excel entregado, se realizó el proceso de análisis y normalización hasta la Tercera Forma Normal (3FN), con el objetivo de eliminar redundancias y garantizar la integridad de la información.

## Tecnologías utilizadas

- PostgreSQL
- Docker
- DBeaver CE
- Draw.io
- GitHub

## Estructura del proyecto

- Script DDL
- Scripts de carga de datos
- Scripts CRUD (INSERT, UPDATE y DELETE)
- Consultas SQL
- Modelo Entidad-Relación (MER)
- Archivo Excel original
- Evidencias de ejecución

## Consultas implementadas

- Stock disponible por producto.
- Movimientos de inventario por bodega.
- Total comprado por proveedor.
- Cantidad de movimientos por bodega.
- Producto con mayor volumen de compras.
